# MAC - QT Environment Setting (in VS Code)

## Install QT
1. Homebrew 설치 
    * 설치 방법 
        - 해당 명령어 터미널에 입력: "/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    * 설치 확인 
        - 해당 명령어 터미널에 입력: "brew --version" 
        - 명령어 입력 후, 버전 명이 표기되면 올바르게 설치된 것
